开始我的PHP学习之路，慢慢学习。

这个计数器坚持使用fopen函数制作，不过还是打开关闭了文件两次，不知道怎么简化才好。

``` {line="1" lang="PHP"}
/*
*counter
*一个基于文本文件的简单计数器
*v0.1
*2005-06-14
*arong
*不保留权利
*/
$countFile="ab.txt";
function displayCounter($cFile){
$fp = fopen($cFile,"rb");
$countNum = fgets($fp);
if($countNum==""){
$countNum = "0";
}
$countNum += 1;
echo("您是第".$countNum."个访问的客人");
fclose($fp);
$fp = fopen($cFile,"wb");
fwrite($fp,$countNum);
fclose($fp);
}
displayCounter($countFile);
```

这个计数器的另一个版本，晕，居然用了3次打开文件操作。没办法，菜鸟就是这样的。

``` {line="1" lang="PHP"}
/*
*counter
*一个基于文本文件的简单计数器
*v0.1
*2005-06-01
*arong
*不保留权利
*/
$countFile="ab.txt";
function displayCounter($cFile){
if(!file_exists($cFile)){
$fp = fopen($cFile,"w");
fwrite($fp,"0");
fclose($fp);
}
$fp = fopen($cFile,"r+");
$countNum = fgets($fp);
fclose($fp);
$countNum += 1;
echo("您是第".$countNum."个访问的客人");
unlink($cFile);
$fp = fopen($cFile,"w");
fwrite($fp,$countNum);
fclose($fp);
}
displayCounter($countFile);
```
